### 1. 问题描述（简洁明确）
```
【当前问题】
项目已完成基础开发，但存在以下问题需要优化：

1. 原有3个视觉风格中，“几何”和“水彩”两种风格的视觉效果不够美观
2. 需要替换为5种新的、更酷炫的动态风格
3. 新风格必须与音乐节奏高度适配
```

### 2. 具体修改要求（核心部分）

#### 2.1 风格调整
```
【风格调整方案】
保留：
- ✅ 唯美·星河流转（效果不错，继续使用）

移除：
- ❌ 酷炫·几何图形（不美观）
- ❌ 治愈·水彩晕染（不美观）

新增5种风格（请按以下顺序实现）：
```

#### 2.2 新增5种风格的详细描述（关键！）

```
1. 【风格名称】极光幻境
   - 视觉特征：流动的极光幕布效果，色彩在天空中渐变、波动
   - 节奏适配：低频时极光缓慢波动，高频时出现闪烁光点
   - 色彩方案：青绿+紫罗兰+粉红（可随时间渐变）

2. 【风格名称】粒子火焰
   - 视觉特征：无数发光粒子组成跳动的火焰，随音乐燃烧、升腾
   - 节奏适配：音量越大火焰越高，鼓点时粒子爆炸式扩散
   - 色彩方案：红橙黄渐变，加入少量蓝色形成对比

3. 【风格名称】水晶脉冲
   - 视觉特征：3D旋转的水晶几何体，表面有脉动的光纹
   - 节奏适配：重拍时水晶碎裂并重组，节奏快时旋转加速
   - 色彩方案：透明质感+彩虹色光泽

4. 【风格名称】神经网络
   - 视觉特征：发光的节点和连接线，形成动态网络结构
   - 节奏适配：节点随节拍跳动，连接线随旋律流动
   - 色彩方案：霓虹蓝+紫+荧光绿

5. 【风格名称】气泡星河
   - 视觉特征：无数彩色气泡从底部升起，在顶部破裂
   - 节奏适配：鼓点产生新气泡，音量决定气泡大小
   - 色彩方案：马卡龙色系（粉蓝、粉红、淡紫）
```

### 3. 技术实现建议（给AI的提示）

```
【技术实现要点】
- 继续使用现有的Three.js框架
- 每种风格作为一个独立的类/模块，便于维护
- 保持60fps的流畅度，粒子数量控制在2000-5000
- 确保每种风格都有明确的节奏映射逻辑：
  * 低频(bass) → 控制基础形态/颜色
  * 节拍(beat) → 触发特效
  * 音量(volume) → 控制运动强度
- 提供风格切换的过渡动画（淡入淡出）
```

### 4. 验收标准

```
【验收标准】
1. 5种新风格视觉效果惊艳，无明显卡顿
2. 每种风格都能明显感受到与音乐的同步
3. 风格切换流畅，不会中断音乐播放
4. 代码结构清晰，注释完整
5. 原有“星河流转”风格不受影响
```
